To‑Do List Web App
A simple, responsive To‑Do List web application built using HTML, CSS, and JavaScript only. This project demonstrates how to build an interactive front‑end web app without any external libraries or frameworks. Objective
The goal of this project is to create a lightweight and visually appealing To‑Do List that helps users manage daily tasks efficiently. It stores data locally in the browser using localStorage, so tasks remain saved even after refreshing the page.

File	Description
index.html	Contains the structure of the To‑Do List web app.
styles.css	Defines the layout, colors, and responsive design.
script.js	Handles all the interactive functionality and Local Storage logic.

How to Run

Download or clone this repository.
Ensure all three files (index.html, styles.css, script.js) are in the same folder.
Open index.html in any modern browser.
Start adding tasks and managing your list.

Simple, elegant, and functional — a perfect beginner‑friendly project for learning front‑end web development!
